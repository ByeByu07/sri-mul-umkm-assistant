# Project Title
**SRI MUL**

# Description
**SRI MUL** adalah chatbot assistant khusus untuk membantu individu bahkan pemilik bisnis UMKM juga. 

Anggap seperti [ChatGPT](https://chatgpt.com/) tapi bisa mengingat produk kalian, transaksi kalian bahkan membantu mengurus pembayaran kalian juga. 

Penipuan berkurang karena sudah **terintegrasi dengan payment gateway**, sehingga orang awam mudah sekali untuk mengecek pembayaran apakah sudah terselesaikan atau belum.

Transaksi dan produk kalian akan selalu sama dan tentu didukung juga oleh pelaporan dengan visual UI yang menarik serta UX yang mudah.

Jadi, jangan ragu untuk chat dengan SRI MUL, **SRI MUL selalu disisimu…**.

## Tech Stack:
- nextjs
- tailwindcss
- typescript
- better-auth
- drizzle-orm
- shadcn/ui
- tanstack-query
- claude-code
- ai vercel SDK

# Setup Instructions
- clone this repository

- install all package/library with
```
npm install
```

- To run this project, you will need to setup the following environment variables.

- copy the `.env.example` to `.env.local` or `.env`.
- then fill the required variables.


- see the schema in **drizzle/schema.ts**, that the schema setup with initial for better-auth.
- then :
```bash
npx dk:g
```
- update the `POSTGRES_URL` in `.env`
```bash
POSTGRES_URL=postgres://localhost:5432/{name}
```
- migrate better-auth to local and remote
``` bash
npm run db:m:local
npm run db:m:remote
```
- after changes schema, generated schema with
```bash
npx dk:g
```
- then start the local development with
```
npm run dev
```
> [!WARNING] 
> if u want to use plugin from better-auth, add schema manually then generate and migrate.
